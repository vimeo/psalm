# TaintedSession

Emitted when user-controlled input can be passed into the `$_SESSION` array.

## Example

```php
<?php

$usrname = $_GET["usrname"];
if (!isset($_SESSION["attr_user"])) {
    $_SESSION["attr_user"] = $usrname;
}
```
