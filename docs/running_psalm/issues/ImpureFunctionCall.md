# ImpureFunctionCall

Emitted when calling an impure function from a function or method marked as pure.

```php
<?php

function impure(array $a) : array {
    /** @var int */
    static $i = 0;

    ++$i;

    $a[$i] = 1;

    return $a;
}

/** @psalm-pure */
function filterOdd(array $a) : array {
    return impure($a);
}
```
